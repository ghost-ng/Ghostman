Getting Started

Installation

Requirements: An OpenAI API key or compatible AI service

  1. Download from GitHub Releases
  2. Install Python 3.12+ and dependencies
  3. Run: python -m ghostman
  4. Configure your API key in Settings (see API Setup below)

First Run

  1. Launch Ghostman from your desktop or command line
  2. The application will start in system tray mode
  3. Right-click the tray icon and select "Show Avatar" to open the main interface
  4. Click the gear icon to open settings and configure your API key

API Setup

‚ö†Ô∏è Important: You need an AI service API key to use Ghostman's features.

OpenAI Setup (Recommended)

  1. Visit OpenAI Platform
  2. Create an account or sign in
  3. Go to API Keys section and create a new key
  4. Copy the API key
  5. In Ghostman, click the gear icon ‚Üí AI Model
  6. Paste your API key in the API Key field
  7. Click "Test Connection" to verify

Other AI Providers

  • Anthropic Claude: Direct API support with native integration
  • Google Gemini: Native support for Gemini 2.5 Pro/Flash models
  • Local AI: Ollama, LM Studio with OpenAI-compatible APIs
  • OpenRouter: Access to multiple models through one API

üí° Pro Tip: Use the model profile dropdown to quickly switch between preset configurations. Each profile automatically sets the correct temperature, max tokens, and endpoints for optimal performance.

Troubleshooting

Frequently Asked Questions

‚ùì Ghostman won't start or crashes immediately

Try these solutions:

  • Ensure Python 3.12+ is installed
  • Run: python -m pip install -r requirements.txt
  • Check logs in: %APPDATA%/Ghostman/logs/ (Windows)
  • Try: python -m ghostman --debug for verbose output

‚ùì API connection fails

Common API issues:

  • Verify your API key is valid and has credits
  • Check internet connectivity
  • Try different model profiles (preset configurations)
  • For Anthropic: Get key from Anthropic Console
  • For Google: Get key from Google AI Studio

‚ùì Temperature not supported error

If you see "Temperature not supported by this model":

  • This is normal for some models (GPT-5/o1 reasoning models)
  • Switch to a preset model profile for automatic temperature
  • For Claude models: Use temperature 0.0-1.0 range
  • For OpenAI models: Use temperature 0-2 range
  • For reasoning models: Temperature is fixed at 1.0

‚ùì Conversations not saving

Database troubleshooting:

  • Check if you have write permissions to the app data folder
  • Restart the application to reinitialize the database
  • In Settings ‚Üí Advanced, try "Reset Database"
  • Check available disk space

Log Files

Log files can help diagnose issues:

Log Locations

Windows: %APPDATA%\Ghostman\logs\
macOS: ~/Library/Application Support/Ghostman/logs/
Linux: ~/.config/Ghostman/logs/
